# フルオーダーテンプレート

## 概要

フルオーダーテンプレートは、ページ構成が事前に定義されていない完全カスタム用テンプレートです。

## 特徴

- **ページ数自由**: 必要なページを自由に追加可能
- **site.jsonでナビゲーション管理**: ヘッダー・フッターのリンクはsite.jsonから動的に生成
- **コンポーネント再利用**: 他テンプレートのコンポーネントを自由に組み合わせ可能

## 用途

- 独自の構成が必要な企業
- 特殊な業種・サービス
- オリジナリティを重視するプロジェクト

## 使い方

1. `template-fullorder` をクローン
2. `data/site.json` の `navigation` セクションを編集してナビゲーションを設定
3. 必要なページを `src/app/` 以下に作成
4. 各ページのコンポーネントを実装

## site.json のナビゲーション設定例

```json
{
  "navigation": {
    "main": [
      { "label": "ホーム", "href": "/" },
      { "label": "サービス", "href": "/service" },
      { "label": "会社概要", "href": "/about" },
      { "label": "お問い合わせ", "href": "/contact" }
    ],
    "footer": [
      { "label": "プライバシーポリシー", "href": "/privacy" },
      { "label": "利用規約", "href": "/terms" }
    ],
    "cta": {
      "label": "お問い合わせ",
      "href": "/contact"
    }
  }
}
```

## サンプルサイト

フルオーダーテンプレートは完全カスタム用のため、ショーケースサイトではテンプレートの説明ページのみを表示します。
